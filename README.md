# Soberana Solutions modular

Version modularizada desde preview.html sin cambiar contenido, estilos ni logica.

## Estructura

- `index.html`: marcado principal y referencias a CSS/JS.
- `src/styles.css`: estilos extraidos del bloque `<style>` original.
- `src/data.js`: datos, traducciones y configuracion de textos.
- `src/app.js`: renderizado, filtros, comparador, modal e interacciones.
- `paneles.html`: pagina separada para la seccion de paneles solares.
- `baterias.html`: pagina separada para la seccion de baterias.
- `paquetes.html`: pagina separada para combos solares y paquetes comerciales.

Abre `index.html` directamente en el navegador para ver la pagina.

- `inteligentes.html`: pagina separada para accesorios inteligentes, clima y carga vehicular.
- `database/`: base local JSON y scripts SQL para Supabase.

- `aviso-privacidad.html`: aviso de privacidad base para México.
- `terminos-condiciones.html`: términos y condiciones base para México.
